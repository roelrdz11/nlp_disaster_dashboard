NLP Disaster Tweet Dashboard

Roel Rodriguez

Boston College, Woods College: Applied Analytics, ADAN 7431: Natural Language Processing

Abstract

This dashboard presents a Logistic Regression model trained on disaster tweets to predict if a tweet is related to a disaster or not. The model now also incorporates metadata (keywords and location) in the prediction. This project demonstrates the utility of NLP and metadata in identifying disaster-related content in social media data.

Methods

Preprocessing: The text data was preprocessed by:

  • Removing URLs, special characters, and punctuation.
  • Converting all text to lowercase.
  • Tokenizing the text into individual words.
  • Removing stopwords to focus on key terms.
  • Lemmatizing each word to reduce words to their base form.

Feature Extraction: TF-IDF was applied to the text, and metadata features (keywords and location) were encoded and added.
